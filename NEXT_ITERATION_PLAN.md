# Dashboard Next Iteration Plan

This plan captures the next feature batch for the personal ministry dashboard.

- Add custom calendar event types with a saved list and trash controls for user-created types. Built-in types stay protected.
- Keep warning popups visible above modal blur layers.
- Apply start/end date and time range rules consistently to Add Event, Draft Calendar Block, and Add Schedule.
- Improve mobile Scheduler height and move Morning/Afternoon/Evening labels closer to the top of each slot.
- Reduce headline size in RSS reader modal and split-screen reader.
- Open news articles in the in-dashboard reader, with split-screen support.
- When split-screen reader is active, clicking RSS/news should load directly into the split pane.
- Add split-screen back/forward history.
- Bold Greek/Hebrew verse verbs in `#6F7D5C`.
- Expand vocabulary with curated local cache from OpenScriptures Hebrew and MorphGNT/SBLGNT-derived data, starting small for performance, licensing, and correctness.
- Keep sports tabs in one row on mobile.
- Add right-edge transparent scroll segment navigation for hero, World Watch, Daily Brief, and bottom.
